1. 프로그램 소개
═══════════════════════════════════════════════════════════════

네이버 블로그에 자동으로 포스팅을 작성하고 발행하는 Windows용 
데스크톱 애플리케이션입니다.

✨ 주요 기능:
  • 다중 계정 지원
  • 정확한 예약 발행
  • 스마트 태그 관리
  • URL 자동 수집
  • AI 콘텐츠 자동 생성
  • 실시간 로그 표시


═══════════════════════════════════════════════════════════════
2. 시작하기
═══════════════════════════════════════════════════════════════

[설치]
1. NaverBlog.exe 파일을 원하는 폴더에 저장합니다
2. 파일을 더블클릭하여 실행합니다
3. Windows Defender 경고가 나오면:
   → "추가 정보" 클릭
   → "실행" 버튼 클릭

[시스템 요구사항]
  • Windows 7/10/11 (64비트)
  • 메모리: 최소 4GB RAM
  • 저장공간: 최소 500MB
  • 안정적인 인터넷 연결


═══════════════════════════════════════════════════════════════
3. 계정 관리
═══════════════════════════════════════════════════════════════

[계정 추가하기]
1. 프로그램 상단의 "계정 관리" 섹션을 찾습니다
2. "계정 추가" 버튼을 클릭합니다
3. 네이버 ID와 비밀번호를 입력합니다
4. "저장" 버튼을 클릭합니다

[로그인하기]
1. 계정 목록에서 사용할 계정을 선택합니다
2. "브라우저 열기" 버튼을 클릭합니다
3. 열린 Chrome 브라우저에서 수동으로 로그인합니다
4. 로그인이 완료되면 브라우저가 자동으로 종료됩니다
   (브라우저를 직접 닫지 마세요!)

[계정 삭제하기]
1. 삭제할 계정을 선택합니다
2. "계정 삭제" 버튼을 클릭합니다

💡 팁: 여러 계정을 추가하여 번갈아가며 사용할 수 있습니다


═══════════════════════════════════════════════════════════════
4. 포스팅 작성하기
═══════════════════════════════════════════════════════════════

[수동 모드로 작성하기]
1. "수동" 탭을 선택합니다
2. 제목을 입력합니다
3. 본문 내용을 입력합니다
4. 태그를 입력합니다 (선택사항, 쉼표로 구분)
5. 카테고리를 선택합니다 (선택사항)

[본문 작성 팁]
  • HTML 태그를 사용할 수 있습니다
  • 줄바꿈은 자동으로 처리됩니다
  • 이미지 URL을 입력하면 자동으로 이미지가 삽입됩니다


═══════════════════════════════════════════════════════════════
5. 예약 발행
═══════════════════════════════════════════════════════════════

[예약 시간 설정하기]
1. 우측의 "예약 발행 설정" 섹션을 찾습니다
2. 날짜를 선택합니다
3. 시간을 선택합니다 (HH:MM 형식)
4. "시작" 버튼을 클릭합니다

[발행 과정 확인하기]
  • 하단의 로그 창에서 실시간 진행 상황을 확인할 수 있습니다
  • 상태 표시줄에서 현재 진행률을 확인할 수 있습니다

[발행된 URL 확인하기]
  • 발행이 완료되면 자동으로 URL이 수집됩니다
  • 수집된 URL은 프로그램 하단에 표시됩니다

💡 팁: 예약 시간은 현재 시간보다 최소 1분 이후로 설정하세요


═══════════════════════════════════════════════════════════════
6. AI 콘텐츠 생성
═══════════════════════════════════════════════════════════════

[OpenAI API 키 설정하기]
1. "AI 설정" 탭을 선택합니다
2. OpenAI API 키를 입력합니다
3. "저장" 버튼을 클릭합니다

※ API 키 받는 방법:
  1. https://platform.openai.com 방문
  2. 로그인 후 API Keys 메뉴 선택
  3. "Create new secret key" 클릭
  4. 생성된 키를 복사

[AI로 콘텐츠 생성하기]
1. "AI" 탭을 선택합니다
2. 주제나 키워드를 입력합니다
   예: "봄철 여행지 추천"
3. 생성 옵션을 선택합니다:
   • 어조: 격식체/반말체
   • 길이: 짧게/보통/길게
4. "콘텐츠 생성" 버튼을 클릭합니다
5. 생성된 콘텐츠를 확인하고 수정합니다
6. 만족스러우면 "발행" 버튼을 클릭합니다

💡 팁: AI가 생성한 콘텐츠는 반드시 검토 후 사용하세요

7. 문제 해결
═══════════════════════════════════════════════════════════════

[프로그램이 실행되지 않아요]
→ Windows Defender가 차단했을 수 있습니다
  1. "추가 정보" 클릭
  2. "실행" 버튼 클릭
→ 관리자 권한으로 실행해보세요

[로그인이 안 돼요]
→ 브라우저가 열리면 수동으로 로그인하세요
→ 2단계 인증이 있다면 인증을 완료하세요
→ 로그인 완료 후 브라우저를 직접 닫지 마세요

[발행이 실패해요]
→ 네이버 로그인 상태를 확인하세요
→ 인터넷 연결을 확인하세요
→ "브라우저 초기화" 버튼(🔧)을 눌러보세요

[브라우저가 응답하지 않아요]
→ "브라우저 초기화" 버튼(🔧)을 클릭하세요
→ 프로그램을 재시작하세요

[로그를 확인하고 싶어요]
→ 프로그램 폴더의 "logs" 폴더를 확인하세요
→ app.log 파일에 상세한 로그가 기록됩니다

8. 주의사항
═══════════════════════════════════════════════════════════════

⚠️ 반드시 지켜주세요:

1. 네이버 이용약관 준수
   • 과도한 자동화는 계정 제재의 원인이 될 수 있습니다
   • 하루 발행 횟수를 적절히 조절하세요

2. 개인정보 보호
   • 본인의 계정 정보만 사용하세요
   • 계정 정보는 안전하게 보관됩니다
   • 프로그램을 종료해도 로그인 정보는 유지됩니다

3. 저작권 준수
   • AI로 생성된 콘텐츠도 저작권 검토가 필요합니다
   • 다른 사람의 콘텐츠를 무단으로 복사하지 마세요

4. 프로그램 사용 중
   • 발행 중에는 브라우저를 직접 조작하지 마세요
   • 여러 작업을 동시에 실행하지 마세요
   • 안정적인 인터넷 연결을 유지하세요

5. 백업
   • 중요한 콘텐츠는 별도로 백업하세요
   • 프로그램 업데이트 전 데이터를 백업하세요
